# Weather App

a simple weather application that shows real-time weather details for any city.

---

## features

- search weather by city name
- real-time weather data
- shows temperature, humidity, and weather condition
- clean and responsive user interface

---

## tech stack

- html
- css
- javascript
- python (flask)

---

## note

do not forget to add your api key in the backend `.env` file.
